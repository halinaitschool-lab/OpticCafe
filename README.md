# Optic Café

Static website for Optic Café — salon optyczny w Poznaniu.

## Structure

- `index.html` — homepage
- `styles.css` — styles
- `script.js` — scripts
- `project/` — offer detail pages

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8765
```

Then visit `http://localhost:8765`.
